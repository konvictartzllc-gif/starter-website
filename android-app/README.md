# Dex Android App

This is the Android Studio project for Dex.

## Open It

1. Open Android Studio.
2. Choose `Open`.
3. Select this folder:
   `c:\Users\RACUser\starter website\android-app`
4. Let Android Studio sync the Gradle project.

## Project Defaults

- App name: `Dex`
- Package: `com.konvictartz.dex`
- Min SDK: `26`
- Target SDK: `34`
